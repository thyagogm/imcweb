# Calculadora de IMC 🚀

Site simples para **calcular o IMC**. Basta informar peso e altura, e o resultado aparece na hora, junto com a classificação.

---

## Tecnologias 🛠️

- HTML5
- CSS3
- JavaScript

---

## Como usar ⚡

1. Clone ou baixe o projeto:
```bash
git clone https://github.com/thyagogm/imcweb.git
```
2. Abra `index.html` no navegador.
3. Digite seu peso e altura para ver seu IMC.
